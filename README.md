# 砖拽 转专 注专转 - Hebrew Typing Games

Three fun HTML5 games to improve your Hebrew typing skills!

##  Games Included

### 1.  Run and Jump (专抓 拽驻抓)
- Press the correct Hebrew key to jump over obstacles
- Each obstacle shows a Hebrew letter - type it to jump
- 3 lives to complete as many jumps as possible
- Difficulty: Easy

### 2.  Hebrew Car (转 注专转)
- Navigate left, center, and right using random Hebrew keys
- Avoid obstacles and collect diamonds
- Keys change each game for variety
- Difficulty: Medium

### 3. 锔 Typing Sentences (拽转 砖驻)
- Type sentences from Hebrew newspapers
- Three difficulty levels: Easy, Medium, Hard
- Track WPM (words per minute) and accuracy
- 10 sentences per game
- Difficulty: Hard

##  Deployment to Vercel

This project is ready to deploy to Vercel:

1. Install Vercel CLI:
```bash
npm install -g vercel
```

2. Login to Vercel (use credentials from ~/pret):
```bash
vercel login
```

3. Deploy:
```bash
cd /mnt/c/PycharmProjects/hebrew_game
vercel
```

4. Follow the prompts to complete deployment

##  Features

- **Pure HTML5/CSS3/JavaScript** - No dependencies required
- **Responsive Design** - Works on desktop and mobile
- **RTL Support** - Proper right-to-left Hebrew text direction
- **Beautiful UI** - Modern gradient design with smooth animations
- **Progressive Difficulty** - Games get harder as you progress
- **Score Tracking** - Track your performance in each game
